# E-Commerce Customer Purchase Analysis

## Project Overview
This project analyzes customer purchase data from an e-commerce platform to extract meaningful business insights using SQL, Excel, and Python.

## Problem Statement
The objective is to analyze transaction data to identify:
- High-value customers
- Most popular products
- City-wise revenue distribution

## Technologies Used
- SQL
- Microsoft Excel (Pivot Tables, Formulas)
- Python (Pandas, Matplotlib)

## Dataset
Fields included:
- customer_id
- customer_name
- city
- product
- quantity
- price

## Implementation

### SQL
- Created table and inserted data
- Performed aggregation queries:
  - Total spending per customer
  - Product popularity
  - City-wise revenue

### Excel
- Created Total Purchase column
- Built Pivot Table for product sales
- Analyzed city-wise revenue

### Python
- Data loading using pandas
- Created total_purchase column
- Performed:
  - Customer spending analysis
  - Product sales analysis
  - City-wise revenue analysis
- Generated bar chart visualization

## Key Insights
- Sneha is the top customer by transaction value
- Laptop and Mobile generate highest revenue
- Mumbai contributes the highest revenue
- High-value purchases dominate the dataset
